Mini Todo App
Функции
- Добавление задач
- Отметка задач как выполненных
- Удаление задач
- Автофокус на поле ввода после добавления задачи

Запуск локально:
1. Клонируем репозиторий:
git clone https://github.com/alexVinter/mini-todo.git
cd mini-todo
npm install -g serve
serve -s . -l 80
2. открываем в браузере:
http://localhost:80
